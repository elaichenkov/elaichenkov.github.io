# Yevhen Laichenkov's Blog

Personal blog built with [Astro](https://astro.build/) using the AstroPaper architecture.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Content

Posts are stored in:

- `src/data/blog/*.md`

## Deployment

GitHub Pages deployment is handled by:

- `.github/workflows/deploy.yml`
